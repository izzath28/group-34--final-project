# SportSphere Elite Node

SportSphere is a premium sports venue management and booking platform designed for the modern athlete and arena owner. Built with a focus on seamless user experience, high-performance architecture, and institutional-grade standards.

## 🚀 Key Features

### For Players (User Portal)
- **World-Class Search**: Find and book sports courts (Cricket, Football, Badminton, etc.) across Sri Lanka.
- **Instant Confirmation**: Real-time slot booking with zero hassle.
- **Sports Memories**: Upload and share photos from your matches.
- **AI Assistant**: A dedicated AI bot to help you find venues and answer your queries.
- **Secure Sessions**: 1-hour session validity for enhanced data protection.

### For Venue Owners (Center Portal)
- **Subscription Management**: Choose from Base Utility, Elite Pro, or Institutional tiers.
- **Premium Mock Gateway**: Integrated payment systems supporting Card, Bank Transfer, and "Pay Later" (7-day deadline).
- **Venue Management**: Easy-to-use interface to update venue details, pricing, and header images.
- **Reservation Tracking**: Real-time overview of all student/player bookings and revenue analytics.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism & Vibrant Aesthetics).
- **Backend & Auth**: Firebase Integration (Authentication, Firestore, Storage).
- **Icons**: Font Awesome 6.4.0.
- **Typography**: Inter & Outfit (Google Fonts).

## 🔒 Security & Performance
- **Session Expiry**: Implemented a robust 1-hour session check for both users and centers.
- **Data Integrity**: Local session awareness on the landing page for frictionless transitions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 📥 Getting Started
1. Clone the repository: `git clone <repo-url>`
2. Configure your Firebase settings in `js/firebase-config.js`.
3. Open `index.html` in your favorite browser.
