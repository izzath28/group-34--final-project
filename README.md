# SportSphere | The Digital Guardian of Athletic Infrastructure

SportSphere is an institutional-grade sports venue management and health-informed booking ecosystem. Designed for the modern elite athlete and arena owner, it bridges the gap between commercial efficiency and personal health safety.

## 🌟 Major Innovation (UCSC Special)
### **Health-Informed Booking Ecosystem**
Unlike traditional booking platforms, SportSphere introduces a health-first approach to sports:
- **Integrated Health Dashboard**: A sophisticated tool for users to input biometrics (Age, Weight, Height) and receive instant physical assessments.
- **BMI Analysis**: Automatic calculation and categorization of Body Mass Index.
- **Cardio Safety Zones**: Algorithmic calculation of Target Heart Rate zones to prevent overexertion during intense matches.
- **Smart Recommendations**: The system intelligently recommends suitable sports based on the user's current physical profile.
- **Major Clinical Reporting**: Generates professional PDF progression reports with historical data tables for long-term health tracking.

## 🚀 Key Features

### For Players (User Portal)
- **Elite Booking Interface**: Find and book world-class venues (Cricket, Football, Badminton, etc.) with real-time slot validation.
- **Cloud Memory Vault**: A Firebase-powered digital archive to store and cherish match-day photos and memories.
- **AI Smart Assistant**: An on-page AI bot (SSB Engine) providing instant support and sports-related queries.
- **Payment Flexibility**: Support for instant card payments or "Pay Later" options at the venue.

### For Venue Owners (Center Portal)
- **Growth Analytics**: Real-time revenue tracking and performance monitoring.
- **Institutional Management**: Scalable infrastructure handling, from slot management to tiered subscriptions.
- **Automated Check-ins**: Seamlessly manage player logs and reservation statuses.

## 🛠️ Technology Stack
- **Architecture**: Single Page Application (SPA) logic using Modern Vanilla JavaScript.
- **Styling**: Vanilla CSS3 with **Glassmorphism** and High-Aesthetic Vibrant Dark Mode.
- **Backend Infrastructure**: Full **Firebase** Integration:
    - **Authentication**: Secure Google & Email/Password sign-in.
    - **Firestore**: Real-time NoSQL database for bookings and health records.
    - **Cloud Storage**: Highly scalable storage for high-resolution sports memories.
- **Reporting Engine**: `jsPDF` and `jspdf-autotable` for professional document generation.

## 🔒 Security & Standards
- **99.9% Uptime Goal**: Engineered for institutional reliablity.
- **Session Protection**: Robust 1-hour rolling session expiry for data privacy.
- **PCI Compliance Ready**: Mock payment gateway following standard financial security flows.

## 📥 Getting Started
1. **Clone**: `git clone <repo-url>`
2. **Config**: Add your Firebase API keys to `js/firebase-config.js`.
3. **Launch**: Open `index.html` via Live Server or any browser.
